# Pokemon Themed Calculator
**Disclaimer:** This project is for educational purposes.

**Features:**
- Addition (+)
- Subtract (-)
- Multiply (×)
- Divide (÷)
- Convert to Fraction (1/x)
- Square (x²)
- Square Root (√)
- Convert to Percent (x/100)
- Handles Decimals
- Toggle (+/-)

## Usage
- To launch calculator, open from index.html
- Strictly input numbers with buttons to perform operations.
